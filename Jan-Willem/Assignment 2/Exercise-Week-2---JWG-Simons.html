Defining the task

The randomly generated task of interest is that of comparing the graph features of a social network simulated under a social network model to the same graph features from an empirical social network. The three model candidates are the ErdsRnyi, Barabsi-Albert and Watts-Strogatz models, each of which have historically been used to describe and understand real-world social network structures. As we will see some graph features can indeed be captured by these models. Generally speakin ghowever, these models are not able to provide a valid description of real-world social networks. Even so, from the tradition of these models social network researchers ultimately transitioned to other models which are better able to capture the structure of real-world social networks, such as exponential random graph models (ERGMs) and stochastic actor-oriented models (SAOMs).

## Defining the function.
describe <- function(g) { # Function with as its input a graph "g";
  return(c(density = edge_density(g), # calculate density; 
           transitivity = transitivity(g,type='global'),  # calculate transitivity; 
           deg.variance = var(degree(g)), # calculate degree variance; 
           av.geo.dist = mean_distance(g,unconnected=FALSE))) # calculate average geodesic distance.
}

Network simulation

Here I generate a list of 1000 undirected Bernoulli graphs which have the same size as the empirical network. I specify the tie probability to be equal to the observed density of the empirical network.

### Generating a list of 1000 undirected ErdsRnyi graphs with size equal to empirical 
### network and empirical network density as Bernoulli distributed probability for forming 
### a tie.
## Store number of vertices in object "size".  
size <- vcount(net.symm)
## Store empirical density in object "dens'. 
dens <- edge_density(net.symm) 
## Draw 1000 random undirected ErdsRnyi graphs. 
bernoulli1000 <- lapply(1 : 1000, function(i){
  erdos.renyi.game(n = size, p.or.m = dens, directed = FALSE)
  })

Here I generate another list of 1000 undirected networks with a Barabsi-Albert model such that the average density matches the observed density. Here the networks are also of the same size as the empirical network.

### Generating a list of 1000 undirected Barabsi-Albert graphs with size and mean density equal to empirical 
### network.
## Sum over rows of empirical adjacency matrix.We need to observe that vertices enter one after the other in this 
## model. We therefore calculate this sequence, where we make sure that we only count ties that are added to
## nodes that already have entered before. 
sequence <- rowSums(as_adjacency_matrix(net.symm, type = "lower", sparse = FALSE)) 
## Apply barabasi.game() function 1000 times and store resulting scale-free graphs in a list. 
scalefree1000 <- lapply(1 : 1000, function(i){
  barabasi.game(n = size, out.seq = sequence, directed = FALSE)
  })

Here I generate a third list of 1000 undirected networks with a Watts-Strogatz model such that again the average density matches the observed density, but also so that the average clustering (transitivity) matches the observed clustering (transitivity). Again, the networks are of the same size as above.

### Generating a list of 1000 undirected Watts-Strogatz graphs with size equal to empirical network and mean density 
### and transitivity close to equal to empirical network.
## Function that modifies the sample_smallworld function of the igraph package such as to generate a graph with 
## desired numbers of nodes and edges by "thinning".
my_small_world <- function(n,e,p) {
  half.degree <- ceiling(e / n)  
  net <- sample_smallworld(dim = 1, size = size, nei = half.degree, p = p)
  net <- delete_edges(net, sample(E(net), ecount(net) - e)) 
  return(net)
}
## Empirical parameters to calibrate to:
# Empirical number of edges. 
nr.edges <- ecount(net.symm) 
# Empirical number of vertices. 
size <- vcount(net.symm) 
# Empirical transitivity.
trans <- transitivity(net.symm)
## Define function for which the zero defines the rewiring parameter that gives empirical transitivity:
mean.trans.smallworld <- function(p) {
  mean(sapply(1 : 1000, function(i){ 
    return(transitivity(my_small_world(n = size, e = nr.edges, p = p)))})) - trans}
## Find this parameter (takes a while):
p.for.transitivity <- uniroot(mean.trans.smallworld, lower = 0, upper = 1)
## Generate the desired sample of networks:
smallworld1000 <- lapply(1 : 1000, function(i){
  net <- my_small_world(n = size, e = nr.edges, p = p.for.transitivity)
})

Some notes on what this means

We can see that the empirical position of the transitivity and average geodesic distance properties of the empirical network are located in the density distribution of the Watts-Strogatz model. For the degree variance this empirical value is situated in the density distribution of the ErdsRnyi random graph model. This tells us that the Barabsi-Albert scale-free graphs do not seem to capture the structural properties of the empirical network at all, whereas the ErdsRnyi random graphs capture at least the degree variance property, and the Watts-Strogatz small world graphs capture both the transitivity and average characteristic path length properties but not the degree variance property.

It is not surprising that the small-world network provides the best description of the empirical network, because it explicitly tries to strike a balance between clustering and characteristic path length (which is an important property of empirical networks), whereas the other two do not (random tie formation in the first, preferential attachment in the second). It is somewhat surprising that the degree variance is best captured by the ErdsRnyi random graph, with the variance in degree being very low in the Watts-Strogatz small world graphs and much too high in the Barabsi-Albert scale-free graphs. It makes sense that the variance will be high in the Barabsi-Albert graphs, due to the preferential attachment mechanism some nodes will have many ties and most only a few to none. Upon thinking on it it also makes sense that the degree variance is low in the Watt-Strogatz small world graph, because most vertices are clustered with only a few having linking ties to other clusters. The degree variance of the empirical network lies somewhere in between these two, which apparantly is best captured by the ErdsRnyi random graph, i.e., by Bernoulli distributed randomness.
